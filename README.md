# Feature Extraction and Steering for Enhanced Chain-of-Thought Reasoning in Language Models(EMNLP 2025)



This is the code for paper "Feature Extraction and Steering for Enhanced Chain-of-Thought Reasoning in Language Models". 

Due to time limit, we have only released the core code, and the complete code will be updated later. The core code is divided into four parts:

1:SAE-based steering: ``` sae_gsm.py```

2:SAE-free steering: ```free_gsm.py```

3:compute eigenvectors: ```eigenvec.ipynb```

4:LLM_judge_prompt: ```llm_judge.ipynb```

Most of the core implementation processes of the paper are covered in the above files, and the remaining necessary codes will be updated as soon as possible.
